# oil-price-predictor

## Project Structure
```
|-- checkpoint      # Model Weights 
|-- data            # Dataset of Oil Price and Factors
|-- model           # Implementations of Prediction Algorithms
|-- Preprocessor    # Preprocess Raw Data
|-- reportGen       # Generate Reports for the Predicted Results
|-- script          # Prepare Args for main.py
|-- dataLoader.py   # Tools for Loading Raw Data
|-- main.py         # Activate the Whole System
```
